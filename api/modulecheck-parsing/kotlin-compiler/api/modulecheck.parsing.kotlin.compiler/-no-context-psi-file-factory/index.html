NoContextPsiFileFactory

class NoContextPsiFileFactory @Inject constructor : AbstractMcPsiFileFactory, McPsiFileFactory(source)

A simple McPsiFileFactory which performs no source code analysis and has just a generic coreEnvironment and configuration. The resulting BindingContext will always be "empty" and will be useless for type resolution.
